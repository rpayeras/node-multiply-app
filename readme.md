# Muliply App with Node Js

```
Options:
      --help     Show help                                             [boolean]  
      --version  Show version number                                   [boolean]  
  -b, --base     A base number to multiply                   [number] [required]  
  -l, --list     Show a list of multiply on console                    [boolean]  
  -t, --to       Limit of multiplications                               [number]
```
